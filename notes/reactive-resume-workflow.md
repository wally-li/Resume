# Reactive Resume Workflow

## 日常维护

1. 在 Reactive Resume 中编辑简历。
2. 导出 PDF、JSON，必要时导出 DOCX。
3. 用最新导出文件覆盖 `exports/` 中的同名文件。
4. 如果这版用于投递，复制一份到 `versions/`，文件名包含日期和方向。

## 推荐版本名

- `versions/2026-05-frontend.pdf`
- `versions/2026-05-frontend.json`
- `versions/2026-05-company-name-frontend.pdf`

## 注意事项

- JSON 是最重要的备份文件，可以重新导入 Reactive Resume。
- PDF 是最终投递文件。
- 不要提交包含身份证、家庭住址等敏感信息的版本。
